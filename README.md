# 🌍 Countries of the World

A simple React web app for visualizing all the countries in the world.

<img width="2034" height="1176" alt="Screenshot from 2026-03-09 22-18-20" src="https://github.com/user-attachments/assets/7ff02c3c-5442-45e4-8b43-ec4b3fd58154" />

## Built With

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [REST Countries API](https://restcountries.com/)
- [Tailwind CSS](https://tailwindcss.com/)

## How to run

```bash
# Clone the repo
git clone https://github.com/angelobroere/countries-of-the-world.git

# Install dependencies
cd countries-of-the-world
npm install

# Start the dev server
npm run dev
```
